# dota2-web-api-wrapper

## Installation
  `npm install dota2-web-api`

## Usage
  ```javascript
  var Dota2API = require('@kurlp00/dota2-web-api-wrapper');
  
  var dota2API = new Dota2API('yourkey', 'language');```
  
## Functions
  
  **Get Heroes** 
  
  ```
  dota2API.getHeroes(function(res) {
    console.log(res);
  );
  ```
